# Packard Promo — Tanzania's Printing, Signage & Promotional Products Platform

Packard Promo is the dedicated print, branding, signage, packaging and promotional products platform of Packard Limited.

## Tech

- Next.js 16 (App Router) + TypeScript
- Tailwind CSS v4
- Framer Motion
- AI Business Tools (client-side estimators)

## Brand

- Same Packard brand system as the parent: navy + gold + glass-dark UI
- Sub-brand identity: "Packard Promo" — focused on print, signage, promo products

## Local dev

```bash
npm install
npm run dev
```

Visit http://localhost:3000
